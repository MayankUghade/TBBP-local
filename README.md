# Big Brother Project

## Setup

1. Install dependencies

```bash
pnpm install

// or use npm
npm install
```

2. Run Next.js project

```bash
pnpm dev

// or use npm
npm install
```

## Stack

- [Next.js](https://nextjs.org/docs/app/)
- [Mantine UI - v7](https://mantine.dev/)
- CSS Modules
- TypeScript
